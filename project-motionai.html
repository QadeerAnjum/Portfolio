Project Overview

Motion AI is an innovative Flutter-based mobile application that harnesses the power of artificial intelligence to transform text prompts and images into stunning video content. Built with a Python FastAPI backend, the app seamlessly integrates with cloud-based video generation APIs to create visually compelling videos in real-time.

The app features a sophisticated monetization system using Google Play Billing for both in-app purchases and subscription models, demonstrating expertise in mobile commerce, API integration, and AI technologies.

Flutter Dart Python FastAPI AI/ML Google Play Billing RESTful API Provider

App Screenshots

01

Home & Prompt Input

Clean and intuitive interface for entering text prompts. Users can describe their vision, select video duration, style preferences, and quality settings. The AI analyzes the prompt and suggests optimal parameters for best results.

02

Image + Prompt Mode

Advanced feature allowing users to upload an image and combine it with a text description to generate creative videos. Perfect for adding motion to still images, creating story sequences, and bringing static concepts to life.

03

Video Gallery & Preview

Organized gallery of all generated videos with thumbnails, creation dates, and quick actions. Built-in video player for preview, share functionality to social media, and download options. Each video is stored locally and synced to cloud storage.

04

Premium Subscriptions

Elegant subscription management interface integrated with Google Play Billing. Users can choose between monthly or yearly plans, view benefits, manage subscriptions, and track usage. Free users get limited credits with clear upgrade prompts.

05

Credit System

Flexible credit-based monetization where users can purchase packs of video generations. Different pricing tiers for varying video lengths and quality settings. Transaction history, credit balance display, and promotional offers create an engaging user experience.

Key Features

Text-to-Video AI

Advanced natural language processing converts detailed text descriptions into high-quality video content with accurate scene interpretation.

Image Input Support

Combine uploaded images with prompts to create dynamic videos, perfect for product demos, storytelling, and creative projects.

Google Play Integration

Fully integrated with Google Play Billing for secure subscriptions and one-time purchases with automatic receipt validation.

FastAPI Backend

High-performance Python backend using FastAPI for real-time video generation requests with async processing and queue management.

Cloud API Integration

Seamless integration with cloud-based AI video generation services for scalable and reliable video production.

Social Sharing

One-tap sharing to Instagram, TikTok, YouTube, and other platforms with optimized video exports and built-in watermarking.
