Project Overview

FlexUp is a comprehensive fitness application designed to help users achieve their fitness goals through customized exercise plans, personalized diet recommendations, and AI-powered guidance. The app combines Flutter's cross-platform capabilities with Python backend services and machine learning algorithms to deliver a truly personalized fitness experience.

Built as part of the Computer Science curriculum at COMSATS University, this project demonstrates expertise in mobile development, backend services, artificial intelligence, and database management.

Flutter Dart Python FastAPI Machine Learning MongoDB TensorFlow OpenAI API

Key Features

AI-Powered Personalization

Machine learning algorithms analyze user behavior and preferences to continuously improve workout and meal recommendations.

Intelligent Chatbot

24/7 AI assistant that answers questions, provides motivation, and offers expert guidance on fitness and nutrition topics.

Offline Mode

Core features available offline, allowing users to track workouts and meals without internet connection.

Real-time Sync

Seamless data synchronization across devices using MongoDB Atlas cloud database for consistent experience everywhere.

Barcode Scanner

Quick food entry using barcode scanning with access to comprehensive nutrition database for accurate calorie tracking.

Exercise Library

Extensive library of exercises with HD video demonstrations, step-by-step instructions, and proper form guidance.
